## ğŸ“– Usage

### As Web Background
```html
<div class="wood-pattern oak">
  <div class="wood-grain"></div>
  <div class="wood-texture"></div>
  <div class="wood-knot knot1"></div>
  <div class="wood-knot knot2"></div>
  <div class="varnish"></div>
</div>
```

### Customize Colors
Modify the CSS gradient in the style section:
```css
.custom-wood {
  background: linear-gradient(90deg, 
    #yourcolor1 0%, 
    #yourcolor2 50%, 
    #yourcolor3 100%
  );
}
```

### Extract Single Pattern
Copy any wood card and its associated CSS class to use individually in your project.

## ğŸ¨ Customization Options

### Adjust Grain Intensity
```css
.wood-grain {
  opacity: 0.5; /* Change from 0.3 to desired value */
}
```

### Modify Knot Size
```css
.knot1 {
  width: 60px;  /* Adjust size */
  height: 60px;
}
```

### Change Varnish Effect
```css
.varnish {
  background: linear-gradient(
    135deg,
    transparent 0%,
    rgba(255,255,255,0.2) 50%, /* Adjust opacity */
    transparent 100%
  );
}
```
